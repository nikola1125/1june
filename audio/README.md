# Background music — Alban Skenderaj · VASHA

The site is set to play **`alban-skenderaj-vasha.mp3`** on loop.

## Add the file

1. Get the MP3 on your Mac (export, purchase, or a file you already own).
2. Rename it to: **`alban-skenderaj-vasha.mp3`**
3. Put it in this folder (`audio/`).

Full path:

```
childrens-day-gift/audio/alban-skenderaj-vasha.mp3
```

4. Open or refresh the site and tap anywhere once — the song should start.

If your file is already named something like `Alban Skenderaj- VASHA.mp3`, either rename it as above or change `musicSrc` in `script.js` to match your filename exactly (e.g. `"audio/Alban Skenderaj- VASHA.mp3"`).
